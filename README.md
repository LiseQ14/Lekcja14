# DataTables – Lista studentów

## Opis
Projekt prezentuje tabelę HTML opartą o wtyczkę **jQuery DataTables**.
Dane studentów są wczytywane z pliku JSON.

## Funkcjonalności
- sortowanie kolumn
- wyszukiwanie
- paginacja
- zmiana liczby wierszy (10 / 25 / 50)
- wczytywanie danych z pliku JSON
- dodatkowe kolumny (email, kierunek)
- przycisk „Usuń” – usuwa wiersz z tabeli
- przycisk „Edytuj” – edycja wieku
- eksport danych do CSV i PDF

## Jak uruchomić
1. Sklonuj repozytorium:
https://github.com/LiseQ14/Lekcja14.git
